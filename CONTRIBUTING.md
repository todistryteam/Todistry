# Contributing to ToDistry

Thank you for considering contributing to the ToDistry project! We welcome improvements, bug fixes, documentation, and feature suggestions.

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork locally:
   ```bash
   git clone https://github.com/your-username/todistry-*.git
   ```
3. **Create a new branch** for your changes:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Commit** your changes with clear messages:
   ```bash
   git commit -m "Add: description of change"
   ```
5. **Push** your branch and submit a Pull Request:
   ```bash
   git push origin feature/your-feature-name
   ```

## Guidelines

- Follow the coding style used in the project.
- Keep pull requests focused and small.
- Document any new code or functionality.
- Link related issues when submitting a PR.

Thanks for contributing!
