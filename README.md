# Todistry

Where Legacy Lives, and Family Stories Connect.

Todistry is a modern ancestry and family connection platform designed to bring family trees to life through stories, media, events, and secure collaboration across generations.

## Core Features
- Interactive family tree builder
- Media-rich profiles and legacy cards
- Private and role-based family collaboration
- Event and reunion planning tools
- Secure memory vault for documents, letters, and recordings
- Cross-platform support (Web, iOS, Android)
